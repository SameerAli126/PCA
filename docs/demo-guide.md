# Demo Guide

## Suggested Demo Flow

1. Open the public atlas and show category filtering
2. Search for a seeded facility such as `Lady Reading Hospital`
3. Open the facility detail page and show provenance
4. Log in as `manager@civicatlas.test`
5. Show the dataset catalog and import workflow
6. Upload a small CSV, validate it, and publish it
7. Return to the public atlas and show the new facility appearing
